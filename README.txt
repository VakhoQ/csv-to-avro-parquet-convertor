Simple Generic CSV to Avro/Parquet convector

Test Files:
- Input: CSV files of Expedia Hotel Recommendations
- Input Url: https://www.kaggle.com/c/expedia-hotel-recommendations/data
- Output: Avro, Parquet files.


Build:
- mvn clean package

Usage:
- AvroDataWorker to convert SVN to Avro.
- ParquetDataWorker to convert SVN to AVRO
